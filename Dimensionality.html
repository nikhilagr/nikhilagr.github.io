Methodology

  • Data Preprocessing and Dimensionality Reduction
    1. The data was not in image format but it was represented as 887 dimensional vector. It means each image had 887 features. But as all features will not be good to predict objects in the images applied various dimensionality reduction and feature selection algorithms to select best features for classification.
    2. The first step was to standardize(Z-Score normalization) the the vectors and apply various dimensionality reduction algorithms such as Principal Component Analysis(PCA) and Singular vector decomposition(SVD).
    3. Also, tried various feature selection techniques such as Recursive Feature Elimination, RFE with Cross validation to choose optimal features for classification.
  • Training, testing, and evaluating ML Model.
    1. Trained 8 different ML Models. Best results were obtained by models which use ensemble methods.Choose voting classifier which considered results from 3 different Algorithms (MLP, RandomForest, ExtraTreeClassfier) and classified image with majority voting mechanism.
    2. Got F1 Score of 0.85
  • Comparing Model Performance and latency.
    1. Model evaluation measures such as F1 score, accuracy, train time, time to predict were compared before applying dimensionality reduction and after applying Dimensionality reduction.
    2. Results: Got same f1 score with selecting only 29 features from 887 features. Thus memory footprint reduced by 90% and train time was reduced by 60%

Challenges

  1. Choosing number of features after reduction such that the model performance is not hampered.
  2. Fine tuning model parameters such number of estimators, maximum number of features for Random Forest and ExtraTreeClassifier.

Key Learining

Supervised learning algorithms, Ensemble Methods, Neural Network (Multi layer perceptron model), Dimensionality Algorithms such as PCA and SVD.
